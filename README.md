Trabajo final 2ª Evaluación Desarrollo de Interfaces.

Para este proyecto, primero creamos una aplicación en Angular con varios componentes reutilizables como la cabecera, pie de página, selector de fecha y tarjetas de actividad. Organizamos la interfaz para que sea fácil alternar entre las páginas de "Actividades" y "Monitores". Luego, implementamos la funcionalidad para mostrar bloques de actividades, donde los libres permiten añadir nuevas, y los ocupados tienen opciones para editar o borrar. Usamos Bootstrap para los estilos, dándole un toque moderno y funcional. Finalmente, configuramos las rutas para navegar entre las secciones y probamos todo para asegurarnos de que el diseño es fiel al ejemplo. ¡Listo para gestionar el gimnasio!
